## 📖 About

**Max Muscle Coaching** is a full-stack fitness coaching platform combining a cross-platform Flutter mobile app with a robust Node.js/Express REST API. Track workouts, follow personalized coaching programs and monitor your progress — all in one app.

## ✨ Features

- 🏋️ Personalized workout programs
- 📊 Real-time progress tracking
- 🔐 Google Sign-In + JWT authentication
- 🎮 Gamification system
- 📱 Cross-platform (iOS & Android)
- 🐳 Dockerized backend

---

## 🛠️ Stack

| Layer | Technology |
|---|---|
| Mobile | Flutter, Dart, GetX |
| Auth | Google Sign-In, JWT |
| Backend | Node.js, Express |
| Database | MongoDB / Mongoose |
| DevOps | Docker |
