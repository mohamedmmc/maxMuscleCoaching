# Max Muscle Coaching — Backend

## Quick Start

1. Configure environment variables in `.env` (DB + JWT).
2. Install dependencies: `npm install`
3. Start the server: `npm start`

## Seeds

- Import exercises: `npm run seed:exercises`
- Generate templates: `npm run seed:templates`

- API endpoints: `docs/API.md`
